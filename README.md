# Drashti Shah - Portfolio Website

A highly creative, modern portfolio website built with HTML, CSS, and JavaScript. The site includes both dark and light modes, smooth animations, and a responsive design that feels premium and sleek.

## Features
- **Dynamic Dark/Light Mode:** Seamlessly toggle between vibrant dark mode and clean light mode.
- **Glassmorphism Design:** Premium UI elements with blurred glass effects.
- **Creative Animations:** Interactive hover effects, typing animations, and scroll-reveal cards.
- **Performant & Lightweight:** Built using vanilla web technologies, optimized for speed.
